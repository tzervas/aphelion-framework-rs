# Aphelion Framework (Rust)

A Rust AI framework focused on fast, traceable, and reproducible model building with deep customization.

## Goals
- Transparent, traceable model construction
- High performance with backend pluggability
- Easy-to-use configuration with deep customization
- Clean integration with rust-ai-core, burn, and cubecl

## Status
Early scaffolding. Feature flags are in place for ecosystem integrations.
